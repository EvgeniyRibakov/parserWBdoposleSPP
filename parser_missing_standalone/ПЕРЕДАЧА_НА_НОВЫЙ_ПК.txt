  ИНСТРУКЦИЯ ПО ПЕРЕДАЧЕ НА НОВЫЙ ПК
═══════════════════════════════════════════════════════════════════════════════

📦 ЧТО ПЕРЕДАТЬ:

Передайте ВСЮ папку "parser_missing_standalone" со всеми файлами:

✅ Parser_WB_Missing.py    (основной скрипт)
✅ requirements.txt         (зависимости)
✅ .env_sample              (пример настроек)
✅ README.md                (подробная инструкция)
✅ QUICK_START.txt          (быстрый старт)
✅ INSTALL.bat              (установка зависимостей)
✅ RUN.bat                  (запуск парсера)
✅ ПЕРЕДАЧА_НА_НОВЫЙ_ПК.txt (этот файл)

═══════════════════════════════════════════════════════════════════════════════

🚀 СПОСОБЫ ПЕРЕДАЧИ:

1. USB ФЛЕШКА:
   - Скопируйте папку "parser_missing_standalone" на флешку
   - На новом ПК скопируйте папку в удобное место (например, C:\Parser\)

2. ОБЛАКО (Google Drive, Яндекс.Диск и т.д.):
   - Заархивируйте папку "parser_missing_standalone" в ZIP
   - Загрузите архив в облако
   - На новом ПК скачайте и распакуйте архив

3. ПО СЕТИ:
   - Скопируйте папку по сети на новый ПК

═══════════════════════════════════════════════════════════════════════════════

📋 ЧТО НЕ НУЖНО ПЕРЕДАВАТЬ:

❌ Папка "data/" с результатами (создастся автоматически)
❌ Папка "chrome_parser_profile/" (создастся автоматически)
❌ Файл ".env" (можно создать из .env_sample на новом ПК)

═══════════════════════════════════════════════════════════════════════════════

🖥️ НА НОВОМ ПК:

1. УСТАНОВИТЕ:
   - Python 3.10+ (https://www.python.org/downloads/)
     ⚠ ВАЖНО: При установке отметьте "Add Python to PATH"
   - Google Chrome (https://www.google.com/chrome/)

2. УСТАНОВИТЕ ЗАВИСИМОСТИ:
   - Откройте папку "parser_missing_standalone"
   - Дважды кликните на INSTALL.bat
   - Или в командной строке: pip install -r requirements.txt

3. ЗАПУСТИТЕ ПАРСЕР:
   - Дважды кликните на RUN.bat
   - Или в командной строке: python Parser_WB_Missing.py
   - Следуйте инструкциям на экране

═══════════════════════════════════════════════════════════════════════════════

📝 ПРИМЕЧАНИЯ:

- Все настройки можно изменить в файле .env (создайте из .env_sample)
- Результаты сохраняются в data/missing_articles_results.xlsx
- Подробная инструкция: README.md
- Быстрый старт: QUICK_START.txt
