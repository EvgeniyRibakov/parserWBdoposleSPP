  БЫСТРЫЙ СТАРТ - ПАРСЕР WB MISSING
═══════════════════════════════════════════════════════════════════════════════

📋 ЧТО НУЖНО УСТАНОВИТЬ:

1. Python 3.10+ (https://www.python.org/downloads/)
   ⚠ ВАЖНО: При установке отметьте "Add Python to PATH"

2. Google Chrome (https://www.google.com/chrome/)

═══════════════════════════════════════════════════════════════════════════════

🚀 КАК ЗАПУСТИТЬ (3 ШАГА):

ШАГ 1: Установите зависимости
   Дважды кликните на файл: INSTALL.bat
   Или в командной строке: pip install -r requirements.txt

ШАГ 2: Запустите парсер
   Дважды кликните на файл: RUN.bat
   Или в командной строке: python Parser_WB_Missing.py

ШАГ 3: Следуйте инструкциям на экране
   - Авторизуйтесь в Wildberries
   - Разрешите открытие вкладок в браузере
   - Дождитесь завершения парсинга

═══════════════════════════════════════════════════════════════════════════════

📁 РЕЗУЛЬТАТЫ:

Результаты сохраняются в файл:
   data/missing_articles_results.xlsx

═══════════════════════════════════════════════════════════════════════════════

⚙️ НАСТРОЙКИ (опционально):

1. Скопируйте .env_sample в .env
2. Откройте .env в блокноте
3. Измените настройки при необходимости:
   - HEADLESS_MODE=False  (чтобы видеть браузер)
   - PARALLEL_TABS=20     (количество вкладок одновременно)

═══════════════════════════════════════════════════════════════════════════════

❓ ПРОБЛЕМЫ?

- "python не найден" → Установите Python с галочкой "Add Python to PATH"
- "Chrome не запускается" → Закройте все окна Chrome перед запуском
- "Вкладки не открываются" → Разрешите открытие вкладок в браузере

Подробная инструкция: README.md
